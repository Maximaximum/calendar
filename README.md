# Calendar
To start on your localhost you need to:<br/>

  For backend part:<br/>
    #. install mongodb<br/>
    #. cd backend<br/>
    #. npm install<br/>
    #. npm start<br/>
    
  For frontend part:<br/>
    #. cd frontend<br/>
    #. npm install<br/>
    #. npm start<br/>
 
 Frondend dir has already compiled css flie.<br/>
 Backend will listen port 3000.<br/>
 Frontend will listen port 8000.
